## Путь B — Disasm (верификация)

**Что делать**: вскрыть тела ключевых функций.

**Приоритет:**
```
calculateFullSystem*()  →  bow()  →  NR()
```

**Почему этот порядок**: см. [[research/methodology]]

**Что это даст:**
- Точная φ(v) вместо рабочей гипотезы
- Exact матрицы 1D/2D
- Оптимизации NR (rank-1 / frozen Jacobian)
- Точная реализация Chaos

**Инструменты**: Ghidra / IDA / Binary Ninja
